## Objective
Teach Kalvin that MHALL (Mary Had A Little Lamb) is structurally equivalent to SVO (Subject-Verb-Object), so that Kalvin acknowledges MHALL == SVO and SVO == MHALL as a bidirectional countersign.

## Approach
Introduce identities first, then build the structural decomposition layers using only countersign and undersign (S1-level operators). The final lesson presents the full MHALL == SVO equivalence with subscript scaffolding that Kalvin should ground via fast path.

## Lessons

### 1
Introduce the identity atoms M, H, A, L, S, V, O.

```
M(ary)
H(ad)
A
L(ittle)
S(ubject)
V(erb)
O(bject)
```

### 2
Establish the bidirectional countersign between M and H — two identities that countersign each other.

```
M(ary) == H(ad)
```

### 3
Define the compound identity ALL from A and L via canonization.

```
(A L L)
ALL => A L(ittle) L(ittle)
```

### 4
Establish the structural mapping between grammatical roles and lexical identities using undersign: M undersigns S, H undersigns V, ALL undersigns O.

```
M(ary) = S(ubject)
H(ad) = V(erb)
(A L L)
ALL = O(bject)
```

### 5
Establish the bidirectional equivalence: MHALL == SVO, with subscript scaffolding that decomposes MHALL into its structural components mapped to SVO.

```
(Mary Had A Little Lamb)
MHALL == SVO =>
   S(ubject) = M
   V(erb) = H
   O(bject) = A(ll) =>
     A > D(et)
     L > M(od)
     L > O
```
